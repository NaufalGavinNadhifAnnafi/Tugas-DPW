Kesimpulan: Checkbox tetap bisa dipilih lebih dari satu meskipun memiliki atribut name yang sama.

Kesimpulan: Setiap type pada input memiliki fungsi yang berbeda. Type email akan memvalidasi format email, type password menyembunyikan teks, type date menampilkan pemilih tanggal, dan type number hanya menerima angka.

Percobaan Input Range

Nilai Volume:

Kesimpulan: Input type range digunakan untuk memilih nilai dalam rentang tertentu. Nilai minimum adalah 0 dan maksimum 100.

Percobaan Datalist

Kesimpulan: Elemen datalist memberikan saran pilihan saat pengguna mengetik. Pengguna tetap bisa mengetik manual selain dari daftar yang tersedia.
