Contoh Elemen Audio HTML5

Audio di atas menggunakan atribut controls untuk menampilkan tombol play/pause, autoplay agar audio otomatis berjalan, dan muted agar tidak langsung bersuara saat halaman dibuka.

Kesimpulan: Elemen audio pada HTML5 memungkinkan pemutaran file suara secara langsung tanpa plugin tambahan. Atribut autoplay membuat audio langsung berjalan, namun pada beberapa browser hanya akan berfungsi jika disertai atribut muted.
